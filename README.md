## 기능
- 코랩 또는 런팟 노트북에서 실행
- 코랩 사용시 구글 드라이브 연결
- 런처 설정 초기화, 가져오기, 내보내기
- 저장소, 확장, 모델 등 다운로드
- 확장, 모델, 실행 인자, 커밋 해시 등 즐겨찾기
